# CarbonSense Screenshots

This directory contains screenshots of the CarbonSense India platform showcasing key features and user interfaces.

## Files

- **dashboard.png** - Interactive Tableau dashboard with state-wise emissions data
- **global-view.png** - 3D globe visualization with ozone distribution
- **chatbot.png** - AI-powered chatbot interface for climate queries

## How to Add Screenshots

1. Capture screenshots of the running application
2. Save them in PNG format (recommended resolution: 1280x720 or 1920x1080)
3. Name them descriptively and place in this directory
4. Update the main README.md with new screenshot references if needed

## Screenshot Guidelines

- Use clear, high-resolution images
- Capture meaningful user interactions and features
- Include both desktop and mobile views
- Ensure no sensitive data is visible in screenshots
